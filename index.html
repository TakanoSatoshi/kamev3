Gnikrap

Gnikrap is a simple to use script development environment for the Lego Mindstrom EV3.

Download a released version

Develop within the browser.

You can develop from the device of your choice (Windows, Appel, Linux, Android, etc.), there is nothing to install except a decent browser (Tested on Firefox and Chrome).

A full documentation is available.

The API available is fully documented with code samples. There is operational samples code that are commented.

Browse the documentation

Empower the device' sensors.

With Gnikrap you are able to use the device' sensors (Touch screen, keyboard, gyroscope sensor, video camera) within your script as simply as if they were standard EV3 sensors.

Last but not least Gnikrap is Open Source software.

As Gnikrap is open source, you can consult, modify and redistribute the software (see the licence file for more details).
